## 기술 스택

React, Styled-Component, Vercel

## 주요 기능

롤링페이퍼 생성, 메세지 작성 및 수정, 슬라이더 보기, 이모지 리액션 추가, 카카오톡 공유

## 멤버

- 박지선 (Frontend)
- 김성연 (Frontend)
- 문필겸 (Frontend)
- 우준석 (Frontend)
- 정현진 (Frontend)
